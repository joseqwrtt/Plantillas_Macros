# 📋 Plantillas Macro

> Pega tus plantillas en cualquier aplicación de Windows con solo escribir `/`

Plantillas Macro es una pequeña aplicación para Windows que vive en segundo plano y te permite insertar documentos DOCX, ODT y TXT directamente en cualquier programa — Ticketing, Outlook, el Bloc de notas, tu CRM, campos de texto... donde sea que puedas escribir.
